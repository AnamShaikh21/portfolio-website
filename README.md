# Portfolio Website (Python + Flask)

This is a simple personal portfolio website built using **Python and Flask**.
The project demonstrates basic web development concepts and Git-based version control.

---

## Tech Stack
- Python 3
- Flask
- HTML
- CSS
- Git & GitHub

python -m venv venv
